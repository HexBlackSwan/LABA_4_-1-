# LABA4App (C++/CLI Windows Forms)

Десктопное приложение для учета успеваемости студентов, реализованное как портирование логики консольного CRUD-приложения в Windows Forms.

## Что реализовано
- 2 окна: главное окно (`MainForm`) и окно добавления/редактирования (`EditRecordForm`).
- Перегрузка конструктора во втором окне:
  - `EditRecordForm()` — сценарий добавления (визуально: заголовок/кнопка "Добавить", редактируемый ID).
  - `EditRecordForm(const Record&)` — сценарий редактирования (визуально: заголовок/кнопка "Сохранить", ID только для чтения).
- Загрузка и сохранение данных из/в файл CSV через `OpenFileDialog` / `SaveFileDialog`.
- Добавление/изменение/удаление записей + ручной ввод данных.
- Сортировка (по ID, по имени, по оценке по убыванию).
- Проверка корректности ввода и защитная логика (обязательные поля, диапазоны, проверки выбора строки и т.п.).
- Сохранение классов данных в отдельных файлах (`Record.h`, `RecordManager.h/.cpp`).
- Использование `DataGridView` для визуализации таблицы.

## Сборка
1. Откройте `LABA4App.sln` в Visual Studio 2022.
2. Выберите конфигурацию `Debug|x64` или `Release|x64`.
3. Соберите проект `LABA4App`.

Результат сборки: `bin/<Configuration>/LABA4App.exe`.
